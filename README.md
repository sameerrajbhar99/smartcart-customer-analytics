# SmartCart Customer Analytics

A data analytics and preprocessing project built using Python to analyze customer shopping behavior, spending patterns, and customer demographics.

## Project Overview

This is an unsupervised machine learning and data analytics project focused on cleaning, preprocessing, and analyzing customer data from a retail shopping dataset. It includes feature engineering, missing value handling, outlier removal, and data visualization techniques to generate meaningful business insights.

## Features

- Data Cleaning & Preprocessing
- Missing Value Handling
- Feature Engineering
- Customer Spending Analysis
- Outlier Detection & Removal
- Data Visualization using Seaborn & Matplotlib
- Customer Demographics Analysis

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features

The dataset contains customer-related information such as:

- Income
- Age
- Education
- Marital Status
- Customer Tenure
- Spending on Products
- Number of Children
- Campaign Responses

## Data Preprocessing Steps

### 1. Handling Missing Values
- Filled missing income values using median.

### 2. Feature Engineering
Created new features such as:
- Age
- Customer Tenure Days
- Total Spending
- Total Children

### 3. Data Transformation
- Simplified education categories
- Simplified marital status categories

### 4. Outlier Removal
- Removed unrealistic age and income values.

## Visualizations

The project uses:
- Pair plots
- Distribution analysis
- Customer behavior insights
